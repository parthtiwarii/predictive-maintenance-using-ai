# Predictive Maintenance Using AI

## Overview

This project predicts machine failures using Machine Learning and Deep Learning techniques. The objective is to identify potential equipment failures before they occur, reducing downtime and maintenance costs.

## Features

- Predictive maintenance analysis
- Machine failure prediction
- Data preprocessing and evaluation
- Performance metrics generation
- Precision-Recall Curve
- F1 Score Comparison
- Detection Lead Time Analysis

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Project Structure

data/ → Dataset files

src/ → Source code

report/ → Evaluation reports and plots

notebooks/ → Jupyter notebooks

## Results

The model evaluates machine health using classification metrics and generates:

- Precision-Recall Curve
- F1 Score Comparison
- Detection Lead Time Analysis
- Metrics Summary Report
